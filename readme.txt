Dynamic widget content for single pages and posts.

== Description ==

The widget will only show if you have set some content for it on the current page. Perfect for creating dynamic widgets that only appear when you need it.

== Installation ==

1. Upload the `dynamic-widget-content` directory (directory included) to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
