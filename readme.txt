Dynamic widget content for single pages and posts.

== Description ==

The widget will only show if you have set some content for it on the current page. Perfect for creating dynamic widgets that only appear when you need it. Works with any custom post type as well!

Usage examples:

*   Post summaries
*   Positive and negative points for reviews
*   Index with links for long pages
*   ...

Let us know what you think! Any feedback is welcome.

== Installation ==

1. Upload the `dynamic-widget-content` directory (directory included) to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add the 'Dynamic Widget Content' widget
1. Add content in the 'Dynamic Widget Content' meta box for posts or pages
